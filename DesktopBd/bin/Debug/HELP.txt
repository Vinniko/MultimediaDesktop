1. Аппаратные и программные требования
Для работы с приложением необходимо иметь вычислительную систему следующей базовой конфигурации:
1) процессор: 1.0 ГГц;
2) оперативная память: 1 Гб;
3) свободное место на жёстком диске: 100 Мб;
4) видеоадаптер: 64 Мб;
5) разрешение экрана 1920х1080 пикселей;
6) наличие основных устройств ввода-вывода (мышь, клавиатура, монитор).
Необходимое программное обеспечение:
1) операционная система Windows не ниже версии 7;
2) установленный .NET Framework не ниже версии 4.7.2.
Необходимо обеспечить программное взаимодействие приложения с файловой системой.
Для работы с кодом программы вычислительную систему следующей базовой конфигурации:
1) процессор: 2.0 ГГц;
2) оперативная память: 2 Гб;
3) свободное место на жёстком диске: 2 Гб;
4) видеоадаптер: 64 Мб;
5) разрешение экрана 1920х1080 пикселей;
6) наличие основных устройств ввода-вывода (мышь, клавиатура, монитор).
Необходимое программное обеспечение:
1) операционная система Windows не ниже версии 7;
2) установленный .NET Framework не ниже версии 4.7.2;
3) Visual Studio не ниже версии 2017 года;
4) Установленная система для создания пользовательских приложений WPF.

2. Инструкция по установке 
1. Установка приложения.
1.1. Для работы приложения необходимо установить .NET Framework не ниже версии 4.7.2. Скачать его можно с официального сайта Microsoft и установить с помощью скачанного инсталлера.	
1.2. Чтобы установить приложение на персональный компьютер, достаточно разархивировать архив с приложением в любую удобную директорию на компьютере.

3. Инструкция по эксплуатации
2. Начало работы с приложением.
2.1. Для запуска приложения необходимо запустить исполняющий файл Desktop_.exe.
2.2.1.1. Если это первый запуск приложения, то нажать кнопку «No account?».
2.2.1.2.  В открывшемся окне сделать ввести логин в поле «Login» - логин может включать в себя любые символы и состоять минимум из одного, максимум из восьми символов. Также 
необходимо ввести два раза пароль. Пароль может состоять из любых символов и должен быть не менее четырех, не более шестнадцати символов. Если оба введенных пароля совпадают и такой 
аккаунт не зарегистрирован, все поля будут подсвечены зеленым.
2.2.1.3. Нажать кнопку «Register».	
2.2.1.4. В появившемся диалоговом окне нажать «ОК» или закрыть его.
2.2.2.1. Если аккаунт уже был создан или вы только что создали его, то в поля Login и Password ввести корректные логин и пароль.	
2.3. Если пароль был неправильно введен, можно посмотреть правильность его ввода, поставив галочку в соответствующем поле, правее поля для ввода пароля.	
2.4. Можно запомнить введенные данные (не безопасно), поставив галочку возле надписи «Запомнить меня».

3. Создание ярлыка.	
3.1.Чтобы создать ярлык необходимо нажать на кнопку «Добавить новый ярлык» (белый плюсик в зеленом круге), во всплывающем меню или непосредственно на рабочем столе.	
3.2. В появившемся окне нажать кнопку «Обзор…» в левом верхнем углу. 
3.3. В появившемся диалоговом окне выбрать любой интересующий файл и нажать кнопку «Открыть» или если таковых нет – кнопку «Отмена».
3.3.1. Если необходимо искать файлы по какому-либо фильтру или добавить ярлык для папки. Слева от кнопки обзор необходимо выбрать тип файла\папки, для которого\которой будет добавлен 
ярлык.	
3.4. Для задания собственного имени и картинки ярлыка необходимо поставить галочку в поле «Задать вручную:».	
3.4.1. В поле «Текст ярлыка» ввести желаемое имя ярлыка.	
3.4.2. Для выбора собственной картинки необходимо нажать кнопку «Обзор…» в правой нижней части окна.	
3.4.3. В диалоговом окне выбрать понравившуюся картинку и нажать кнопку «Открыть».	
3.5. Предварительный вид ярлыка можно увидеть перед глазами в правой части окна.
3.6. Для подтверждения добавления ярлыка, необходимо нажать кнопку «Добавить» или для отмены настроек, кнопку «Отмена».
3.7. Когда ярлык будет добавлен, для обращения к файлу, к которому он ссылается, достаточно нажать ярлык левой кнопкой мыши. Файл или приложение запустятся.

4.Изменение ярлыка.
4.1. Для изменения ярлыка необходимо нажать на интересующий ярлык правой кнопкой мыши.	
4.2. В появившемся контекстном меню нажать кнопку «Изменить».
4.3. Для изменения пути ярлыка, в появившемся окне нажать кнопку «Обзор…» в левом верхнем углу. 
4.4. В появившемся диалоговом окне выбрать любой интересующий файл и нажать кнопку «Открыть» или если таковых нет – кнопку «Отмена».
4.4.1. Если необходимо искать файлы по какому-либо фильтру или добавить ярлык для папки. Слева от кнопки обзор необходимо выбрать тип файла\папки, для которого\которой будет добавлен 
ярлык.	
4.5. Для изменения текста ярлыка или его изображения поставить галочку (если не стоит) в поле с названием «Задать вручную:».
4.5.1. В поле «Текст ярлыка» ввести желаемое имя ярлыка.	
4.5.2. Для выбора собственной картинки необходимо нажать кнопку «Обзор…» в правой нижней части окна.	
4.5.3. В диалоговом окне выбрать понравившуюся картинку и нажать кнопку «Открыть».	
4.6. Предварительный вид ярлыка можно увидеть перед глазами в правой части окна.
4.7. Для подтверждения изменения ярлыка, необходимо нажать кнопку «Изменить» или для отмены настроек, кнопку «Отмена».

5.Удаление ярлыка.
5.1. Для удаления ярлыка необходимо нажать на интересующий ярлык правой кнопкой мыши.
5.2. В появившемся контекстном меню нажать кнопку «Удалить».
6. Настройки аккаунта.
6.1. Смена имени пользователя (логина).
6.1.1. Для изменения имени аккаунта необходимо нажать левой кнопкой мыши на кнопку «Настройки» (шестерни) в выпадающем меню или правой кнопкой мыши по имени аккаунта в выпадающем меню 
и в появившемся контекстом меню выбрать соответствующий пункт.
6.1.2. В появившемся окне, в поле «Имя аккаунта» ввести новое имя аккаунта. Имя аккаунта (ваш логин) может включать в себя любые символы и состоять минимум из одного, максимум из восьми 
символов.
6.1.3. Нажать кнопку «Применить» и кнопку «Ок» либо нажать кнопку «Ок». В случае желания сбросить изменения, нажать кнопку «Отмена» или на крестик в правом верхнем углу окна.
6.1.4. В появившемся окне нажать кнопку «ОК».
6.1.5. Перезайти на аккаунт.
6.2. Смена пароля.
6.2.1. Для смены пароля на аккаунте необходимо нажать левой кнопкой мыши на кнопку «Настройки» (шестерни) в выпадающем меню или правой кнопкой мыши по имени аккаунта в выпадающем меню 
и в появившемся контекстом меню выбрать соответствующий пункт. 
6.2.2. В появившемся окне нажать кнопку «Сменить пароль».
6.2.3. В новом окне ввести новый пароль два раза. Пароль может состоять из любых символов и должен быть не менее четырех, не более шестнадцати символов.
6.2.4. Нажать кнопку «ОК» или для отмены кнопку «Отмена». 
6.2.5. Нажать кнопку «Применить» и кнопку «Ок» либо нажать кнопку «Ок». В случае желания сбросить изменения, нажать кнопку «Отмена» или на крестик в правом верхнем углу окна.
6.2.6. В появившемся окне нажать кнопку «ОК».
6.2.7. Перезайти на аккаунт.

6.3. Изменение шрифта наименования ярлыков.
6.3.1. Для изменения шрифта наименования ярлыков необходимо нажать левой кнопкой мыши на кнопку «Настройки» (шестерни) в выпадающем меню или правой кнопкой мыши по имени аккаунта в 
выпадающем меню и в появившемся контекстом меню выбрать соответствующий пункт. 
6.3.2. В появившемся окне нажать кнопку «Изменить» возле надписи «Шрифт».
6.3.3. В диалоговом окне выбрать удовлетворяющие пользователя настройки. Примечание: выбор размера текста, доступен, но не реализован из визуальных соображений.
6.3.4.  Для выбора шрифта и стиля нажать кнопку «ОК» или для отмены выбора – кнопку «Отмена» или же на крестик в правом верхнем углу экрана.
6.3.5. Нажать кнопку «Применить» и кнопку «Ок» либо нажать кнопку «Ок». В случае желания сбросить изменения, нажать кнопку «Отмена» или на крестик в правом верхнем углу окна.
6.3.6. В появившемся окне нажать кнопку «ОК».
6.3.7. Перезайти на аккаунт.
6.4. Выбор изображения рабочего стола (текущего).
6.4.1. Для выбора изображения рабочего стола (текущего) необходимо нажать левой кнопкой мыши на кнопку «Настройки» (шестерни) в выпадающем меню или правой кнопкой мыши по имени аккаунта 
в выпадающем меню и в появившемся контекстом меню выбрать соответствующий пункт. 
6.4.2. В появившемся окне нажать кнопку «Обзор…» возле надписи «Выбрать картинку рабочего стола:».
6.4.3. В диалоговом окне выбрать желаемое изображение.
6.4.4. Для подтверждения выбора изображения нажать «Открыть», для отказа «Отмена».
6.4.5. Выбранную картинку можно увидеть в специальном поле в правой части окна.
6.4.6. Для подтверждения изображения рабочего стола нажать кнопку «Применить» и кнопку «Ок» либо нажать кнопку «Ок». В случае желания сбросить изменения, нажать кнопку «Отмена» или на 
крестик в правом верхнем углу окна.
6.4.7. В появившемся окне нажать кнопку «ОК».
6.4.8. Перезайти на аккаунт.
6.5. Сброс настроек «по умолчанию».
6.5.1. Для сброса настроек аккаунта по умолчанию необходимо нажать левой кнопкой мыши на кнопку «Настройки» (шестерни) в выпадающем меню или правой кнопкой мыши по имени аккаунта в 
выпадающем меню и в появившемся контекстом меню выбрать соответствующий пункт. 
6.5.2. В появившемся окне нажать кнопку «По умолчанию».
6.5.3. Для подтверждения изображения рабочего стола нажать кнопку «Применить» и кнопку «Ок» либо нажать кнопку «Ок». В случае желания сбросить изменения, нажать кнопку «Отмена» или на 
крестик в правом верхнем углу окна. Примечания: картинка сбрасывается только для текущего рабочего стола, а не для всех рабочих столов, которыми владеет пользователь.
6.5.4. В появившемся окне нажать кнопку «ОК».
6.5.5. Перезайти на аккаунт.

7. Руководство пользователя.
7.1. Чтобы посмотреть руководство пользователя необходимо нажать на кнопку «Помощь» (вопросительный знак) в выпадающем меню. 

8. Выход из аккаунта.
8.1. Чтобы выйти из аккаунта, необходимо нажать правой кнопкой мыши на имя аккаунта в выпадающем меню.
8.2. В появившемся контекстном меню выбрать пункт «Выти».

9. Удаление аккаунта.
9.1. Чтобы удалить аккаунт, необходимо нажать правой кнопкой мыши на имя аккаунта в выпадающем меню.
9.2. В появившемся контекстном меню выбрать пункт «Удалить аккаунт». 
9.3. В появившемся диалоговом окне выбрать «ОК» для подтверждения удаления или «Отмена» для её отмены.
9.4. В новом окне нажать «ОК».
9.5. Выйти из аккаунта.

10. Свернуть в трей. 
10.1. Чтобы свернуть приложение в трей, необходимо нажать на соответствующую кнопку в правой верхней части экрана в выпадающем меню. 

11. Закрыть приложение. 
11.1. Чтобы закрыть приложение, необходимо нажать на соответствующую кнопку в правой верхней части экрана в выпадающем меню. 

4. Перечень файлов
Структурно приложение состоит из следующих файлов-классов:
1) Файлы сущностей: UserModel.cs, LabelModel.cs, DesktopModel.cs;
2) Классы форм приложения: AutorizationView.xaml.cs, ChangeLabelView.xaml.cs, CreateLabelView.xaml.cs, DesktopView.xaml.cs, PasswordChangeView.xaml.cs, RegistrationView.xaml.cs, 
SettingView.xaml.cs;
3) ViewModel классы: AutorizationViewModel.cs, ChangeLabelViewModel.cs, CreateLabelViewModel.cs, DesktopViewModel.cs, PasswordChangeViewModel.cs, LabelViewModel.cs, 
RegistrationViewModel.cs, SettingViewModel.cs;
4) Классы репозитории: AccountRepository.cs, DesktopRepository.cs, LabelRepository.cs;
5) Классы сервисы: AutorizationService.cs, DesktopService.cs, FileService.cs, ImageService.cs, LabelService.cs, RegstraitionService.cs, ValidationService.cs, ViewManager.cs;
6) Файлы, в которых хранится вся информация об аккаунтах и их настройках: accountList.dat, MemoryAccount.dat, LOCAL_ACCOUNT_SETTINGS_имяПользователя_DESKTOP.dat.
